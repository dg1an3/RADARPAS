theWheel Configuration Management

The source files and headers associated with theWheel code base can be divided into three groups.  The wlObject base class describes the first layer of extensions to the object base class.  The wlView family describes the main mapping activity.  theWheel family describes the standard application components, including the application class theWheelApp, the frame window wlFrameWnd, and the child window class wlChildWnd.
