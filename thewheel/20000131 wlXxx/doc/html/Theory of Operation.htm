theWheel Technical Specification
Theory of Operation

theWheel operates within the framework of a rigorously defined model of human-computer interactions, referred to as the Standard User Model.  This model defines the human-computer interactions as a pair of observable state vectors, plus state vectors for the user, the user interface, and objects within the user interface.

theWheel then has access to the observable state vectors (the mouse and the monitor), the internal state vector of the user interface, and internal state vectors for objects within the user interface.
